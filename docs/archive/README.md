## Adding to the archive

Move files here only when:
1. The file is **explicitly superseded** by another live doc (link the successor).
2. The file describes a feature/initiative that has been **explicitly cancelled**
   (record the decision in the archived doc's frontmatter).
3. The file is a **point-in-time snapshot** kept for traceability but not
   actionable.

Do NOT archive a doc just because its content is partially out of date —
fix the doc instead.
